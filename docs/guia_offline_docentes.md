# Guia rapida para trabajar sin internet (offline)

Si aparece el mensaje:
"No hay datos en cache. Necesitas conexion para cargar este partido por primera vez"

Significa:
La primera vez que entras a un partido, debes hacerlo con internet para que el dispositivo guarde una copia local.

## Pasos (muy simple)

1. Conectate a internet una vez.
2. Entra a la liga y abre la lista de partidos.
3. Abre los partidos que vayas a usar ese dia.
4. Ya puedes trabajar sin internet.

## Opcion rapida: preparar toda la liga offline

1. Entra a la liga con internet.
2. Pulsa "Preparar liga offline".
3. Espera a que termine.
4. Ya puedes trabajar sin internet, aunque no hayas abierto cada partido.

## Que pasa si no hay conexion

- Los cambios se guardan en tu dispositivo.
- Cuando vuelva internet, se sincronizan solos.

## Si ves el mensaje rojo otra vez

- Pulsa "Reintentar carga" cuando tengas internet.
- O pulsa "Restaurar backup" si tienes un archivo guardado.

## Consejo practico

Antes de ir al aula, abre la lista de partidos con internet.
Asi todos quedan listos para el modo offline.

## Nota importante

La preparacion offline se guarda solo en el dispositivo y navegador donde se realiza.
Si usas otro ordenador o navegador, hay que prepararlo de nuevo.
